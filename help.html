Time

If no colons are used, the time defaults to minutes. For example, entering 23 is the same as entering 00:23:00

If only one colon is used, the time is assumed to be minutes:seconds. For example, entering 23:30 is the same as entering 00:23:30

If the time includes hours, two colons should be used and the total time should be entered like this:

hours:minutes:seconds

For example, for fifteen hours enter:

15:00:00

Distances

Distance will default to miles, but can be entered in the following formats and they will be converted to miles:

  • Kilometers
  • k
  • meters
